#### lm-LSTM
通过训练小说《三国演义》，对给定的文本，预测下一个汉字。<br>
1.对训练数据ETL，生成<N个汉字,N+1汉字>方式；<br>
2.进行独热（one-hot）编码;<br>
3.使用LSTM训练<br>
#### 环境
python3.6 tensorflow1.9

#### 模型训练
```Bash
python lm_lstm.py

```
